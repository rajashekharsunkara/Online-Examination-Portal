❓ TROUBLESHOOTING:

Issue: "Permission denied"
Fix:   newgrp docker

Issue: "Docker not found" after install
Fix:   exec bash   (or logout/login)

Issue: Port already in use
Fix:   sudo lsof -i :5173   (find and kill process)

Issue: Script hangs
Fix:   bash -x ./setup-demo.sh   (debug mode)

More help: cat INSTALLATION.md
